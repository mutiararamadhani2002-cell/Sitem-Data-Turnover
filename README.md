## Project Overview
Tingkat turnover karyawan merupakan salah satu tantangan utama dalam manajemen sumber daya manusia karena berdampak langsung pada stabilitas operasional perusahaan, Proyek ini bertujuan untuk membangun Sistem Data Turnover Karyawan berbasis Python menggunakan Visual Studio Code sebagai lingkungan pengembangan. Sistem ini dirancang untuk membantu perusahaan dalam mengelola data karyawan secara sistematis, memantau status kerja (aktif atau resign), serta menghasilkan laporan analisis sederhana yang mendukung pengambilan keputusan berbasis data.

Pengembangan sistem ini mengikuti alur kerja pemrograman terstruktur, dimulai dari perancangan struktur data menggunakan list dan dictionary, pembuatan fungsi modular untuk setiap menu (Create, Read, Update, Delete, dan Laporan), hingga implementasi validasi input untuk meminimalkan kesalahan pengguna. Selain itu, sistem juga dilengkapi dengan fitur analisis sederhana seperti perhitungan jumlah karyawan resign, tingkat turnover, dan identifikasi alasan resign yang paling dominan. Dengan pendekatan ini, sistem yang dibangun tidak hanya berfungsi sebagai alat pencatatan data, tetapi juga sebagai dasar analisis awal untuk memahami pola turnover karyawan. 

### Problem Statement
Tingginya tingkat turnover karyawan menjadi salah satu permasalahan utama dalam manajemen sumber daya manusia dalam banyak kasus, pencatatan data karyawan dan informasi terkait status kerja masih dilakukan secara manual atau belum terstruktur dengan baik, sehingga menyulitkan proses pemantauan dan analisis.

Ketiadaan sistem yang terorganisir menyebabkan perusahaan kesulitan dalam:
1. Melacak jumlah karyawan aktif dan resign secara akurat
2. Mengidentifikasi alasan utama karyawan keluar
3. Menghitung tingkat turnover secara cepat dan tepat
4. Menghasilkan laporan yang mendukung pengambilan keputusan strategis

Oleh karena itu, diperlukan suatu sistem sederhana yang mampu mengelola data karyawan secara terstruktur, menyediakan fitur pencarian dan pembaruan data, serta menghasilkan analisis dasar terkait turnover karyawan. 

### Objectives
Proyek ini bertujuan untuk merancang dan mengembangkan sistem sederhana berbasis Python yang dapat membantu pengelolaan data turnover karyawan secara terstruktur dan sistematis. Adapun tujuan khusus dari proyek ini adalah:

1. Membangun sistem pencatatan data karyawan 
2. Menyediakan fitur pengelolaan data 
3. Memantau status karyawan (Aktif atau Resign) 
4. Menghitung jumlah karyawan resign dan tingkat turnover 
5. Mengidentifikasi alasan resign yang paling dominan

Berdasarkan hasil pengembangan dan pengujian, Sistem Data Turnover Karyawan berbasis Python berhasil dibangun dan dijalankan dengan baik menggunakan Visual Studio Code sebagai lingkungan pengembangan.

Secara teknis, sistem memanfaatkan struktur data list of dictionary untuk menyimpan informasi karyawan, yang memungkinkan proses pencarian, pembaruan, dan penghapusan data dilakukan secara efisien melalui iterasi dan validasi kondisi. ## Installation and Usage
1. Persyaratan Sistem
- Python versi 3.x
- Visual Studio Code (atau text editor lain)
- Terminal / Command Prompt
2. Langkah Instalasi
Install Python
- Unduh Python dari situs resmi: https://www.python.org
- Pastikan opsi “Add Python to PATH” dicentang saat instalasi.
Install Visual Studio Code (Opsional)
- Unduh dari https://code.visualstudio.com
- Install extension Python untuk memudahkan eksekusi program.
3. Download atau Salin File Program
- Simpan file program dengan nama misalnya:
    
    Sistem_Data_Turnover.py

4. Usage (Cara Menjalankan Program)

- Jalankan Program Buka Visual Code Pythone, lalu arahkan ke folder tempat file disimpan:

    cd Download

- Kemudian jalankan:

    Sistem_Data_Turnover.py
